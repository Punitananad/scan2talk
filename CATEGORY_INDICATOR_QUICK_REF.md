# Category Indicator - Quick Reference

## 🎯 At a Glance

Look at the **top-right corner** of any printed tag:

```
🟢 F  = Free (No recharge needed)
🔵 P  = Prepaid (Wallet recharge required)
🟣 PP = Postpaid (Bill later)
🟠 T  = Trial (Limited free usage)
🔴 D  = Distributor (One-time payment, then free)
```

---

## 📋 Quick Identification

### By Color Only
- **Green** → Free
- **Blue** → Prepaid
- **Purple** → Postpaid
- **Orange/Amber** → Trial
- **Red** → Distributor

### By Letter Only (B&W printing)
- **F** → Free
- **P** → Prepaid
- **PP** → Postpaid
- **T** → Trial
- **D** → Distributor

## 🖨️ How to Use

1. **Generate QR batch** (as usual)
2. **Preview batch** → See colored corners
3. **Download PDF** → Indicators included
4. **Print** → Use color printer for best results
5. **Sort** → Easy to organize by color

## 💡 Pro Tips
