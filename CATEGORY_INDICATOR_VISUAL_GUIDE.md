## 💡 Usage Scenarios

### Scenario 1: Sorting 100 Printed Tags
```
Step 1: Spread tags on table
Step 2: Look at top-right corners
Step 3: Sort into piles:
  - Green pile (Free)
  - Blue pile (Prepaid)
  - Red pile (Distributor)
  - etc.

Time taken: ~2-3 minutes for 100 tags
```

### Scenario 2: Customer Inquiry
```
Customer: "What type of tag do I have?"
You: "Look at the top-right corner. What color do you see?"
Customer: "It's red with a 'D'"
You: "That's a distributor tag. You paid once and it's free forever!"
```

### Scenario 3: Quality Control
```
Before shipping 50 distributor tags:
1. Quick visual scan of all tags
2. Verify all have red "D" in corner
3. If any don't match, investigate
4. Ship with confidence
```

## 🎯 Best Practices

### For Printing
1. **Use color printer** for best results
2. **Print test page** first to check colors
3. **Use 120gsm+ paper** for durability
4. **Check color calibration** on printer

### For Storage
1. **Sort by color** immediately after printing
2. **Label boxes** with category name and color
3. **Store in clear bags** so corner is visible
4. **Keep inventory** by category

### For Distribution
1. **Show customer** the corner badge
2. **Explain meaning** of color/letter
3. **Provide reference card** with color meanings
4. **Take photo** of tag for records
